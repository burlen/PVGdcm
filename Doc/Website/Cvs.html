Using CVS for retrieval of gdcm

To learn more about CVS and to get the CVS programs, please refer to the CVS Home.
  • WARNING
      To avoid troubles users who choose to work with gdcm CVS version are strongly recomended to have a look at the gdcm dashboard *before* doing a 'cvs update'.
      If there is too much red stuff: then DO NOT update.
      Wait a few days until gdcm stabilizes.
      If you like to take risk you can decide to update gdcm if the broken build (red square in 'Build' column) only affect a different plateform than yours.
      Example: If the dashboard is red *only* for the SunOS, and you are using Windows there is a great chance that it would still work for you (also this is not recommended).
